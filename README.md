# dotfiles

`jakan0`'s dotfiles, managed with [`chezmoi`][chezmoi].

Install them with:

```shell
chezmoi init jakan0
```

Or with this, if Chezmoi is not already installed:

```shell
sh -c "$(curl -fsLS https://chezmoi.io/get)" -- -b ~/.local/bin init jakan0
```

[chezmoi]: https://github.com/twpayne/chezmoi
